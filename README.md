# TIVRA – Modern Digital Banking Dashboard

TIVRA is a full-stack FinTech web application designed to provide a seamless, secure, and modern digital banking experience. The platform addresses key challenges of traditional banking systems by introducing automation, real-time updates, role-based access control, and an intuitive user interface. The project was developed as part of the **Infosys Springboard Virtual Internship 6.0**.

## Project Overview

TIVRA offers a comprehensive digital banking solution that enables users to manage accounts, perform secure transactions, pay bills, track spending, and view detailed transaction histories. With a focus on security, scalability, and usability, the application delivers a reliable banking dashboard suitable for modern users.

## Key Features

- Secure user authentication using JWT  
- Role-based access control (Admin & User)  
- Real-time dashboard with account balance and analytics  
- Multi-account creation and management  
- Secure peer-to-peer money transfers  
- Bill payment and budgeting support  
- Complete transaction history with audit trail  
- Live currency converter with real-time exchange rates  
- Responsive and mobile-friendly UI  
- Cloud-based deployment for scalability

## System Architecture

TIVRA follows a modular, client-server architecture:
- The **frontend** handles UI rendering and user interactions.
- The **backend** exposes RESTful APIs for business logic and data processing.
- The **database** ensures reliable and scalable data storage.
- JWT authentication and encryption safeguard user data.
- Modular design enables easy maintenance and future expansion.
